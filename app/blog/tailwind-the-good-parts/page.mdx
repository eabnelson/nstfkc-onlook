# Levels of UI components

- Level -1: Fragments and Layout
- Level 0: Primitive components
- Level 1: Compound components
- Level 2: Containers
- Level 3: Screens (Pages)

## Fragments
Icons, Text, Box, Container

## Core components
Closed internal state. Dense styling. Appearance can be altered via props

Customisation via props

Button, Toggle (Switch), Inputs, Links, Modals

## Compound components
Can't be customised

Forms, Navigation, Product Card

## Containers
Business logic, composition of compund and primitive components

## Screens (Pages)
Composition of containers.
